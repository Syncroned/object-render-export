# Object Render Export

**GD 2.2081 · Geode 5.7.x**  
Windows & Android

Exports selected editor objects as PNG / JPEG / GIF at preset resolutions (720p-4K), preserving editor draw order, colors, opacity, blending and hidden state.

## Usage

1. Select one or more editor objects.
2. Click the **Export** button in the edit tab.
3. Pick a resolution and format.
4. Enable "Transparent BG" for an alpha background (PNG / GIF).

Files are saved to the configured export folder with an auto-generated timestamp.

On Windows, the **Direct Save** setting toggles between saving straight to the configured folder or showing a file picker.
